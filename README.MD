## 项目说明
本项目旨在通过指令微调预训练模型，以提升其在对话文本生成任务中的表现。我们使用了开源的预训练模型(qwen)，并在特定的对话数据集上进行了微调。项目包括以下几个主要步骤：
1. 数据预处理：对原始对话数据格式化，以适应模型的输入要求。
2. 模型微调：使用预处理后的数据对预训练模型进行指令微调。
3. 推理与生成：利用微调后的模型生成对话文本，并评估其质量。 

同时我们实现了作业要求的三个附加任务: batched training, batched inference, beam search.
